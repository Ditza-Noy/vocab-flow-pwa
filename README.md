A mobile-first Progressive Web App (PWA) designed to optimize English vocabulary learning. This application functions entirely client-side, offering a seamless, app-like experience without the need for a backend or active internet connection once installed.

## 📱 Features

* **Multiple Learning Modes:** Includes interactive flashcards, multiple-choice quizzes, and typing practice.
* **Smart Word Selection:** Utilizes a weighted random selection algorithm to prioritize words you need more practice with, optimizing the learning process.
* **Scored Testing:** Track your progress and retention with built-in scored tests.
* **Fully Offline & Installable:** Built as a Progressive Web App (PWA), meaning you can install it directly to your home screen on Android/iOS and use it completely offline.
* **Local Data Management:** All tracks, word lists, and progress are securely saved directly to your device using `LocalStorage`.

## 🛠️ Technology Stack

* **Frontend:** Vanilla JavaScript, HTML5, CSS3
* **Storage:** Browser `LocalStorage`
* **Architecture:** Progressive Web App (manifest.json, Service Worker)

## 🚀 How to Install on Mobile (Android)

Because this is a PWA, you can install it directly from the browser:
1. Open Google Chrome on your Android device.
2. Navigate to the live GitHub Pages URL for this repository.
3. Tap the "Add to Home screen" banner at the bottom of the screen, OR open the three-dot menu in the top right and select "Install app".
4. The app will now appear on your home screen and run in a standalone, full-screen window!

## 💻 Local Development

To run this project locally on your machine:
1. Clone the repository: `git clone https://github.com/ditza-noy/vocab-flow-pwa.git`
2. Open the project folder.
3. Serve the directory using any local web server (e.g., VS Code Live Server, or Python's `python -m http.server`).
4. Navigate to the local port in your browser.
